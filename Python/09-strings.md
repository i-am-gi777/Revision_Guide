A **string** in Python is a sequence of **Unicode characters** enclosed 
in:
- Single quotes `'text'`
- Double quotes `"text"`
- Triple quotes `'''text'''` or `"""text"""` (for multiline)

---



```python
#Creating Strings
a = "Hello"
b = 'Python'
c = """This is
a multi-line string."""

# f-Strings
name = "Abhi"
age = 22

print(f"My name is {name} and I am {age} years old.")
